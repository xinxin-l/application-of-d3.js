# 动态效果有：
1. 点击头像产生高亮效果
2. 点击左上角return刷新页面
3. 点击右上角introduction出现弹窗介绍整个页面
4. 点击玫瑰图的不同部分，相应的点状图条形图都会发生数据变化，同时产生高亮效果
5. 点击页面最下方的年份可以跳转到雷达图界面
